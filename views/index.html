Welcome to my Calorie Buddy Dynamic Web Application

This dynamic web application will function as a digital calorie counter to help you manage your diet.
The dynamic web application interacts with you to calculate and display nutritional facts including calories for your recipes based on food ingredients in the recipe.


Using the navegation bar you can access different functions of the web like register a new user, login, log out etc.
